# Schülerdaten Verwalten

Ein einfaches Python-Projekt zur Verwaltung von Schülerdaten.  
Das Programm ermöglicht das Speichern, Anzeigen, Bearbeiten und Löschen von Schülerinformationen.

---

## Funktionen

- Schüler hinzufügen  
- Alle Schüler anzeigen  
- Schülerdaten bearbeiten  
- Schüler löschen  
- Datenspeicherung (z. B. dateibasiert)  
- Bedienung über die Konsole  

---

## Verwendete Technologien

- Python 3.12  
- Ausschließlich Module aus der Python-Standardbibliothek  

Hinweis: Das Programm wurde mit Python 3.12 getestet.  
Es sollte auch mit anderen Python-3-Versionen funktionieren.
